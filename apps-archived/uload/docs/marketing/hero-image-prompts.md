# Hero Image Prompts für AI-Bildgenerierung

## 🎨 Allgemeine Stil-Richtlinien

**Basis-Stil für alle Prompts:**

- Modern, clean, minimalist design
- Professional business aesthetic
- High-tech, digital atmosphere
- Color palette: Purple (#9333ea), Blue (#3b82f6), White, Light gray
- Soft gradients and glass morphism effects
- Bright, optimistic lighting
- 16:9 aspect ratio for hero sections

---

```
A sleek, modern web dashboard interface floating in 3D space, showing real-time analytics graphs, QR codes, and shortened URLs. Glass morphism design with purple and blue gradients. ```
A dynamic scene showing a long, messy URL transforming into an elegant QR code and short link. Purple to blue gradient flow effect, particles and light trails showing the transformation. ```
Abstract network of interconnected nodes and links floating in space, representing URL connections and analytics data flow. Glowing purple and blue nodes with thin connection lines. Depth of field effect, clean minimalist style, white background with subtle gradient. Tech visualization, data art, modern and professional.

```
Floating iPhone and MacBook mockups showing the uLoad interface, surrounded by floating UI elements, QR codes, and analytics cards. Glass morphism effects, purple accent colors, clean shadows. Devices at slight angles for dynamic composition. Premium product showcase style, Apple-inspired aesthetics.

### 5. Team Collaboration Scene

```
Multiple glass panels floating in 3D space, each showing different user avatars working with links and QR codes. Collaborative workspace visualization, connected by subtle light trails. Purple and blue color accents, modern office aesthetic, clean and professional atmosphere.
```

## 💡 Prompt-Optimierungs-Tipps

### Negative Prompts (was vermieden werden soll):

```
Avoid: cluttered, busy, dark, gloomy, aggressive, outdated, pixelated, stock photo feeling, corporate cliché, generic, boring, flat lighting, oversaturated colors, realistic people, explicit brand logos
```

### Style References:

```
Style inspiration: Stripe.com design, Linear.app aesthetics, Vercel dashboard, Apple product photography, Behance UI/UX showcases, Dribbble trending, modern SaaS marketing
```

### Technical Specifications:

```
Resolution: 2880x1620px (hero section)
Format: PNG with transparency or JPEG high quality
Color space: sRGB
Optimization: Web-optimized, under 500KB
Fallback: Gradient background if image fails to load
```

---

## 🚀 Verwendungshinweise

1. **Testing:** Erstelle 3-5 Varianten pro Konzept für A/B Tests
2. **Lokalisierung:** Bilder sollten kulturell neutral sein
3. **Accessibility:** Ensure sufficient contrast for text overlays
4. **Performance:** Optimize for Core Web Vitals
5. **Responsiveness:** Create variations for mobile/tablet/desktop

## 📝 Beispiel-Workflow für Midjourney/DALL-E

```
/imagine prompt: [Hauptprompt aus obiger Liste] --ar 16:9 --v 6 --style raw --q 2

```
--chaos 10 (für leichte Variationen)
--seed [number] (für reproduzierbare Ergebnisse)
