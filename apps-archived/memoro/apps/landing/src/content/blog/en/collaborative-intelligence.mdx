### 1. Establish Common AI Standards

**The Problem:** Everyone uses AI differently, leading to inconsistent results.

**The Solution: Team Playbook for AI**

```
AI Standards Checklist:
□ Unified prompt templates for recurring tasks
□ Common quality criteria for AI output
□ Defined use cases: When do we use which tool?
□ Transparency rules: When do we label AI usage?
□ Privacy guidelines: What can go into AI tools?
```

**Example Prompt Library:**
```
Meeting Summary:
"Create a summary from this Memoro recording including:
- Participants and date
- 3-5 main discussion points
- Decisions made (with reasoning)
- Action items (Who, What, By when)
- Open questions for next meeting"

Project Status Update:
"Based on meeting notes from the last week:
- Current project status (traffic light system)
- Completed milestones
- Current blockers/risks
- Next steps
- Required resources/decisions"
```

## Best Practices for Team AI

### AI Etiquette in Teams

**The Golden Rules:**

1. **Transparency First**
   - "This draft was created with AI and edited by me"
   - Clearly label AI-generated content

2. **Quality Over Quantity**
   - Don't use AI for everything just because you can
   - Value human expertise

3. **Learn Together**
   - Share successful prompts with the team
   - Regular AI learning sessions

4. **Feedback Culture**
   - "The AI summary missed X"
   - Continuous improvement of prompts

### Privacy and Confidentiality

**Clear Rules Create Trust:**

```
Privacy Matrix:
┌─────────────────┬────────────┬─────────────┐
│ Data Type       │ Memoro OK? │ Other AI?   │

│ Strategy Plans  │ ✓          │ ✗           │
│ Personal Info   │ ✗          │ ✗           │

```

### Quality Assurance for AI Content

**The Four-Eyes Principle 2.0:**

1. **AI Creates** → First draft
2. **Human Reviews** → Facts, tone, relevance
3. **AI Refines** → Based on feedback
4. **Human Finalizes** → Last check

**Quality Gates:**
- Fact checks for important documents
- Tone-of-voice review for external communication
- Compliance check for regulated content
