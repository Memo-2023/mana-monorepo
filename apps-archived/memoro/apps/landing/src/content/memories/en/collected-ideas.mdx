# Collected Ideas & Suggestions

The Collected Ideas Memory captures all creative suggestions and innovative approaches from your conversation.

## Key Features

### Intelligent Idea Recognition
Automatically identifies:
- Explicit suggestions and ideas
- Creative solution approaches
- Improvement suggestions
- Innovative concepts
- Alternative approaches
- Spontaneous inspirations

### Feasibility Categorization
Each idea is categorized by:
- **Short-term** (0-3 months): Quick to implement
- **Medium-term** (3-12 months): Planning required
- **Long-term** (12+ months): Strategic vision

### Evaluation Criteria
- **Impact**: Potential effect
- **Effort**: Estimated resources
- **Innovation**: Novelty level
- **Risk**: Implementation risks
- **Quick Win**: Quick successes

## Use Cases

- Document brainstorming sessions
- Build innovation pipeline
- Develop product roadmap
- Collect process improvements
- Evaluate creative workshops
- Team idea management

## Tips for Optimal Results

1. **Explicitly mark ideas**
   - "I just thought of..."
   - "What if..."
   - "We could..."
   - "An idea would be..."

2. **Provide context**
   - Problem being solved
   - Target audience of the idea
   - Expected benefit
   - Inspiration/source

3. **Discuss feasibility**
   - "We could implement this quickly"
   - "This needs more preparation"
   - "Long-term..."

4. **Include evaluation**
   - "That would be a game-changer"
   - "Quick win with big impact"
   - "Interesting but complex"
