# Über Oliver Wolkenstein

Oliver ist das technische Rückgrat unseres Development-Teams. Mit über 10 Jahren Erfahrung in der Softwareentwicklung bringt er tiefgreifendes Wissen in Cloud-Architekturen und skalierbaren Systemen mit.

## Technische Philosophie

"Code ist wie eine Bergroute - er sollte sicher, effizient und gut dokumentiert sein. Jeder, der nach dir kommt, sollte den Weg verstehen können."

Oliver glaubt an:

- **Clean Code**: Lesbarer Code ist wichtiger als cleverer Code
- **Test-Driven Development**: Keine Feature ohne Tests
- **Continuous Learning**: Technologie entwickelt sich ständig weiter
- **Performance First**: Optimierung von Anfang an mitdenken

## Schwerpunkte bei Memoro

- **Cloud-Infrastruktur**: Design und Implementierung der gesamten Cloud-Architektur
- **Performance**: Sicherstellung, dass Memoro auch bei Millionen von Nutzern schnell bleibt
- **Code Reviews**: Mentor für das Team, Reviews aller kritischen Code-Änderungen
- **Technologie-Evaluation**: Bewertung neuer Tools und Frameworks
- **System-Architektur**: Entwicklung robuster, wartbarer Systemdesigns

## Technischer Stack

**Hauptsprachen**: Go, Rust, TypeScript  
**Cloud**: AWS, Google Cloud, Kubernetes  
**Datenbanken**: PostgreSQL, Redis, DynamoDB  
**Tools**: Docker, Terraform, GitHub Actions

## Persönlicher Hintergrund

Olivers Leidenschaft fürs Bergsteigen spiegelt sich in seiner Herangehensweise ans Programmieren wider: methodisch, sicherheitsbewusst und immer mit dem Gipfel im Blick. Er hat einen Master in Informatik von der TU München und mehrere Cloud-Zertifizierungen.

In seiner Freizeit:

- Besteigt Berge in den Alpen
- Contributet zu Open-Source-Projekten
- Experimentiert mit neuen Programmiersprachen
- Hält Vorträge auf Tech-Meetups

"Bei Memoro kann ich meine zwei Leidenschaften verbinden: eleganten Code schreiben und dabei helfen, ein Produkt zu bauen, das wirklich einen Unterschied macht."
