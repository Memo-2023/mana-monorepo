## Start Any Project

```bash
# Use dev:*:full for the best experience
pnpm dev:chat:full
pnpm dev:picture:full
pnpm dev:quotes:full
pnpm dev:contacts:full
pnpm dev:calendar:full
```

These commands automatically set up databases and start all required services.
