# Project Meeting & Status Update

This blueprint optimizes project documentation and ensures transparent communication with all stakeholders.

## Key Features

### Executive Summary
Compact project status for management with the most important KPIs and decisions.

### Detailed Progress Report
Comprehensive documentation of all work packages, milestones, and deliverables.

### Milestone and Deadline Management
Overview of critical paths, achieved milestones, and upcoming deadlines.

### Risks and Escalations
Identification of obstacles, risks, and necessary escalations.

## Used Prompts

1. **Executive Summary**: Project status executive summary
2. **Detailed Summary**: Detailed progress report
3. **Tasks & Appointments**: Milestones and critical paths
4. **Open Questions**: Risks and escalation needs

## Use Cases

- Steering committee meetings
- Sprint reviews
- Project status reports
- Stakeholder updates
- Gate reviews
- Portfolio meetings
- Agile retrospectives

## Tips for Optimal Results

1. Use project metrics and KPIs
2. Visualize progress with percentages
3. Name risks clearly and distinctly
4. Propose concrete solution approaches
5. Document decisions with reasoning
6. Create clear action recommendations
7. Prepare different scenarios
