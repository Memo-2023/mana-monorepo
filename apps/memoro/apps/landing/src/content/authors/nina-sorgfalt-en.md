# About Nina Sorgfalt

Nina is the guardian of quality at Memoro. As QA Engineer and Test Lead, she ensures that every feature, function, and interaction works perfectly - before it reaches users.

## The Detective of Software Quality

With a Computer Science degree (B.Sc.) and additional training in User Experience Design, Nina has the perfect combination of technical know-how and user focus. Her career began as a software developer, but she quickly discovered her true passion: hunting bugs and perfecting user experience.

"As a developer, I wrote code. As a QA Engineer, I ensure that code improves people's lives - reliably, every day."

## Testing Philosophy

Nina's approach to quality assurance is holistic:

### Core Principles
1. **Prevention over Detection**: Avoid errors, don't just find them
2. **User First**: Every test from user perspective
3. **Automation Smart**: Automate what makes sense
4. **Accessibility Always**: Accessibility is not a nice-to-have

### Her Specialties at Memoro

**1. Voice UI Testing Excellence**
- Development of unique test framework for speech recognition
- Testing in 15+ languages and dialects
- Accent and ambient noise tests
- Emotional speech pattern recognition

**2. Automation with Heart**
- 80% test coverage through intelligent automation
- Continuous testing pipeline
- AI-powered test generation
- Visual regression tests

**3. Accessibility as Standard**
- WCAG AAA compliance tests
- Screen reader compatibility
- One-handed operation tests
- Cognitive accessibility tests

**4. Performance Under the Microscope**
- Real-time transcription latency tests
- Memory optimization for long sessions
- Offline functionality testing
- Cross-device synchronization
