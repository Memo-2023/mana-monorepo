# On-Page SEO Optimierungs-Checkliste für Memoro

## 🚨 Kritische Optimierungen (Sofort umsetzen)

### 1. **Title Tags optimieren**
Die aktuellen Titles sind zu generisch und nutzen nicht die Haupt-Keywords aus dem Keyword-Plan.

**Homepage** (home.mdx)
- **IST:** "Memoro - Sprechen statt Tippen"
- **SOLL:** "Meeting Protokoll Software - Automatische Protokollerstellung | Memoro"
- **Warum:** Integriert Hauptkeyword "meeting protokoll software" und "automatische protokollerstellung"

**Features-Seite** (features.mdx)
- **IST:** "Funktionen | Memoro"
- **SOLL:** "KI-gestützte Transkription & Spracherkennung - Funktionen | Memoro"
- **Warum:** Keywords "transkription" und "spracherkennung" einbauen

**Preise-Seite** (prices.mdx)
- **IST:** "Preise | Memoro"
- **SOLL:** "Preise & Kostenlose Testversion - Meeting Software | Memoro"
- **Warum:** "kostenlose testversion" ist wichtiges Conversion-Keyword

### 2. **Meta Descriptions erweitern**
Aktuelle Descriptions sind zu kurz und nicht optimiert für CTR.

**Homepage**
- **IST:** "Memoro verwandelt gesprochene Worte in strukturierte Dokumentation..."
- **SOLL:** "✓ Automatische Meeting-Protokolle in Minuten ✓ 80+ Sprachen ✓ DSGVO-konform ► Sparen Sie 3+ Stunden/Woche. Jetzt 14 Tage kostenlos testen!"
- **Zeichen:** 155 (optimal)

**Industries-Seite**
- Spezifische Description für jede Branche erstellen mit Keywords wie:
  - "baustellendokumentation app"
  - "vorlesungsmitschrift software"
  - "büro protokoll software"

### 3. **H1-H3 Struktur verbessern**
- Jede Seite braucht genau eine H1 mit Hauptkeyword
- H2s sollten verwandte Keywords enthalten
- Hierarchie muss logisch sein

## 📊 Wichtige Optimierungen (Innerhalb 1 Woche)

### 4. **Neue Landing Pages erstellen**
Basierend auf Keyword-Plan fehlen diese kritischen Seiten:

```
/de/meeting-protokoll-software
/de/automatische-transkription
/de/spracherkennung-deutsch
/de/vergleich/otter-ai-alternative
/de/integration/microsoft-teams
```

### 5. **URL-Struktur optimieren**
- **Problem:** URLs wie `/de/features` sind zu generisch
- **Lösung:** Sprechende URLs wie `/de/funktionen/meeting-protokoll`

### 6. **Interne Verlinkung**
- Keyword-reiche Ankertexte verwenden
- Von Homepage zu allen wichtigen Unterseiten
- Breadcrumbs implementieren

## ⚡ Quick Wins (Heute machbar)

### 7. **Alt-Texte für Bilder**
```markdown
IST: alt="Memoro App"
SOLL: alt="Meeting-Protokoll Software Memoro - Automatische Aufnahme im Büro"
```

### 8. **Schema Markup erweitern**
- SoftwareApplication Schema ✓ (bereits vorhanden)
- FAQPage Schema hinzufügen
- Review/Rating Schema für Testimonials
- LocalBusiness für Impressum

### 9. **Content-Länge erhöhen**
- Homepage: Von ~500 auf 1500+ Wörter
- Features: Detaillierte Beschreibungen pro Feature
- Industries: Case Studies hinzufügen

## 🔍 Technische SEO-Fixes

### 10. **Core Web Vitals**
- Bilder mit `loading="lazy"` versehen
- Kritisches CSS inline einbetten
- JavaScript-Bundle optimieren

### 11. **Mobile Optimierung**
- Touch-Targets vergrößern (min. 48x48px)
- Font-Size minimum 16px auf Mobile
- Viewport Meta-Tag prüfen

### 12. **Sitemap & Robots.txt**
- Sitemap bereits vorhanden ✓
- Robots.txt prüfen und optimieren
- Canonical URLs für alle Seiten

## 🎯 Metriken & Tracking
