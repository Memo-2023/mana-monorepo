## SEO-Technische Requirements

### On-Page Optimierung
- **URL-Struktur**: Kurz, keyword-reich, sprechend
- **Title Tag**: 50-60 Zeichen, Keyword am Anfang
- **Meta Description**: 150-160 Zeichen, CTA enthalten
- **H1**: Ein H1 pro Seite, Keyword enthalten
- **Content-Länge**: Minimum 800 Wörter, Ziel 1500+
- **Interne Links**: 3-5 relevante interne Links
- **Bilder**: Alt-Tags mit Keywords, WebP-Format

### Schema Markup
```json
{
  "@context": "https://schema.org",
  "@type": "SoftwareApplication",
  "name": "Memoro",
  "applicationCategory": "BusinessApplication",
  "offers": {
    "@type": "Offer",
    "price": "0",
    "priceCurrency": "EUR"
  }
}
```

### Performance-Ziele
- Core Web Vitals: Alle im grünen Bereich
- Mobile-First: 100% responsive
- Ladezeit: < 3 Sekunden
- Accessibility: WCAG 2.1 AA
